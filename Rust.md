# Rust

一個靜態、完整 error handling、easy read compile error info、有套件管理、可以自動生成 layout 與 documents。看看我又發現了什麼！

和 [[Python]] 同樣有 type hint(annotating) 與 type inference。而且寫法也很像。

```rust
fn add(x: u32, y: u32) -> u32 {
	return x + y
}
```


可以重復宣告變數。(Rust 稱之為 shadow)
